## 1. Create an Account

Sign up at the [MetaLink Dashboard](https://metalink-web.vercel.app/auth/login) using GitHub or Google.

## 2. Generate an API Key

From the [Dashboard](https://metalink-web.vercel.app/dashboard), click **Create API Key**. Copy the key immediately — it won't be shown again.

Your key looks like: `ml_live_abc123def456...`

## 3. Make Your First Request

```bash
curl "https://metalink-api.fly.dev/v1/metadata?url=https://github.com" \
  -H "X-API-Key: ml_live_YOUR_KEY_HERE"
```

## 4. See the Response

  "site_name": "GitHub",
  "type": "website",
  "locale": "en_US",
  "twitter_card": {
    "card": "summary_large_image",

That's it! Check out [Authentication](/authentication) for more details on API key usage, or the [API Reference](/api-reference/get-metadata) for the full endpoint documentation.
